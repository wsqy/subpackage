工具使用了依赖于python2.7+版本的zipfile模块，所以在使用本工具前请先确保系统自带python2.7+,可以参见附属材料安装pyenv

运行前请安装require中列出的文件，如果已安装pip工具，可以执行 pip install -r require自带全部安装
金山云的SDK（ks3）不能通过pip安装，源码地址为：git clone https://github.com/ks3sdk/ks3-python-sdk.git
ks3依赖于six 模块  可通过 pip install six 安装
ks3可用分块上传   可通过pip install filechunkio 安装

阿里云sdk可通过 pip install oss2 安装


script("git clone https://github.com/ks3sdk/ks3-python-sdk.git /home/qy/ks3")
